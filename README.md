## Installation

To run the application:

Clone the project:

```bash
  git clone https://github.com/tahayeslbg/e-commerce.git
```

Go to the project directory:

```bash
  cd /e-commerce
```

Install required packages

```bash
  npm install
```

To run in development mode:

```bash
  npm run dev
```

To run in production mode:

```bash
  npm run build
  npm run start
```
